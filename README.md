# sp6-2_smart-table_starter

https://github.com/martakozhevina/smart-table.git

реализован функционал таблицы, которая отображает данные о продажах
выведено на страницу окружение таблицы из шаблона;
сформированы и выведены строки из того же шаблона;
вся таблица окружена формой и слушаются события change, submit и reset

реализованы поиск, фильтрация, сортировка и пагинация
